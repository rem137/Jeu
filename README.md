# Chemical Tower Defence

Simple tower defence demo. Open `index.html` in a browser to play.

* Appuyez sur "Commencer" pour lancer la partie.
* Faites glisser une tour de la réserve vers le champ pour la placer si vous avez assez d'or.
* Sélectionnez une tour pour ouvrir un panneau où vous pouvez l'améliorer ou la vendre (le coût y est indiqué). Vous pouvez aussi la vendre rapidement avec **Shift**.
* Utilisez le bouton "Start Wave" pour déclencher une nouvelle vague et "Pause" pour mettre le jeu en pause.
