# Chemical Tower Defence

Simple tower defence demo. Open `index.html` in a browser to play.

* Appuyez sur "Commencer" pour lancer la partie.
* Touchez le champ pour placer une tour si vous avez assez d'or.
* Touchez une tour pour l'améliorer (le coût s'affiche dans l'encart d'info) ou maintenez **Shift** pour la vendre.
* Utilisez le bouton "Start Wave" pour déclencher une nouvelle vague et "Pause" pour mettre le jeu en pause.
